# learn_transformer
transformer原理，各部分代码实现以及transorformer在CV领域的简单例子
本代码库分为两部分。transformer原理部分和OCR例子部分。  

## transformer原理部分：
- 博客部分为transformer初探.md，介绍了transformer原理以及各个模块的实现
- 代码部分在code目录下，为Untitle1.ipynb.
## OCR小例子部分：
- 博客为OCR.md，重点介绍了将transformer引入来解决OCR任务的动机与思路，并结合代码详细介绍了细节
- 代码部分在code目录下，为Untitle2文件夹下的所有py文件
